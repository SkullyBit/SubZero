# SubZero

**SubZero** is an ultra-low-latency trading connectivity library for C and C++. It is designed to support high-performance trading systems with minimal overhead and maximum efficiency.

SubZero is a fork of [Libtrading](https://github.com/libtrading/libtrading), which served as the foundation for this project. SubZero builds upon Libtrading with ongoing improvements and optimizations tailored to modern financial applications.

---

## Features
- High-performance, low-latency design.
- Protocol support for FIX, FIX/FAST, and proprietary protocols such as ITCH and OUCH.
- Modular architecture for trading system integration.
